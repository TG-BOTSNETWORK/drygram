# DryGram

DryGram is a production-grade, fully asynchronous Telegram MTProto framework built for Python 3.13+. Inspired by MTProto v2.0, it is designed from scratch with zero PYROGRAM/TELETHON dependencies.

## Core Pillars
- **Performance**: High concurrency with asyncio, connection pools, and memory optimizations.
- **Originality**: Custom APIs (`deliver`, `echo`, `Gates`, `Watchers`) and markup builders.
- **2026 Features**: Out-of-the-box support for Stars, Upgradable Gifts, Stories, Business Bots, and Group Video Streaming.
