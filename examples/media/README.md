# Placeholder Media Directory

This directory is a placeholder for local media files used in the `examples/calls/` scripts.

To run the voice/video calling examples successfully, please place your own media files here and name them accordingly:
- `sample.mp3`
- `sample.wav`
- `sample.ogg`
- `sample.flac`
- `sample.mp4`
- `sample.webm`
- `sample.mkv`
- `sample.mov`
- `sample.avi`

*(No copyrighted media files are bundled in the repository.)*
